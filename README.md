# DataAnalyst-Project

A collection of data analysis projects demonstrating exploratory analysis, customer segmentation, dashboard design, and student performance modeling.

## Projects Included

- `E-commerce-Customer-Segmentation/`
  - Customer segmentation using K-Means clustering.
  - Includes silhouette analysis and insights into purchase behavior.
  - Contains a Jupyter notebook and supporting data/visualizations.

- `Sales-Dashboard-Using-Excel---Data-Analyst-Project/`
  - Sales performance dashboard built in Microsoft Excel.
  - Analyzes sales by category, state, customer, and monthly trends.
  - Includes raw sales data and dashboard files.

- `Students-Performance-Analytics/`
  - Student grades and performance analytics.
  - Demonstrates feature engineering, visualization, and machine learning classification.
  - Includes Jupyter notebooks and dataset files.

## How to Use

1. Open the subfolder for the project you want to explore.
2. Review the project-specific `README.md` in that folder for details.
3. For notebook-based projects, launch Jupyter or JupyterLab and open the `.ipynb` file.
4. For the Excel dashboard project, open the workbook from the `Sales-Dashboard-Using-Excel---Data-Analyst-Project/` folder.

## Notes

- Each project is self-contained and includes its own documentation.
- The notebooks are best viewed in an environment with Python and Jupyter support.
- The sales dashboard is intended for use with Microsoft Excel.
